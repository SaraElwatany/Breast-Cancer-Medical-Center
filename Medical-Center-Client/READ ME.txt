**Instructions**
- Install the Requirements: pip install -r requirements.txt
- Then, make database migrations: python manage.py makemigrations
- python manage.py migrate

To Run As A Client Side:

- Open New Terminal
- Run the application: python manage.py runserver 8050
- Take the URL that appears, and use it in a Private Window like Mozilla FireFox, or Incognito Window like Chrome to avoide any conflicts with the Server Side
